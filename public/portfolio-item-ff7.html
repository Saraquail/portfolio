Final Fantasy 7 Companion

A full stack app

A passion project

The Unofficial Final Fantasy 7 Companion App is an app to assist players of the game in finding and saving information on monsters encountered during game play.

This app was built using JavaScript, React, CSS. The server was built using Node.js, Express, and Knex. The database was built using PostgreSQL. Testing was done using Supertest, Mocha, Chai, Jest, and Enzyme.
